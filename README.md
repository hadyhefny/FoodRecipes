# FoodRecipes
FoodRecipe application interacts with a webservice (RESTfull Api) using retrofit2 to retrieve food recipes. Retrieved data is stored on a local sqlite database cache using room persistence library. FoodRecipe is designed with MVVM architecture pattern.
